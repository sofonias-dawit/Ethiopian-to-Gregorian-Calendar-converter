This project is a C++ program that converts dates between the Ethiopian Calendar and the Gregorian Calendar. It supports conversion in both directions.

- Convert Ethiopian dates to Gregorian dates
- Convert Gregorian dates to Ethiopian dates
- Simple and fast console-based program

## How to Compile
g++ main.cpp -o calendar_converter

## How to Run
./calendar_converter

## Description
The program uses mathematical formulas to accurately convert dates between the two calendar systems.
